# Zenolve Feeds

Static site for **feeds.zenolve.com** — one landing page per Zenolve public-data
feed (Apify pay-per-result actors), with schema, live sample data, code snippets,
FAQ, and JSON-LD structured data so search engines and AI answer-engines can
discover and cite each feed.

- **Live:** https://feeds.zenolve.com
- **Feeds source:** https://apify.com/zenolvepro

## How it's built

Static HTML, no framework. Pages are generated by
`auto_builder/builds/zenolve-feeds-site/generate.py`, which pulls a live sample
from each feed at build time and renders the pages here. Deploys via GitHub Pages
(root of `main`); the `CNAME` file points it at `feeds.zenolve.com`.

To rebuild: run the generator, copy its `dist/` output here, commit, and push.
