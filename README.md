# Zenolve Feeds

Static site for **feeds.zenolve.com** — one landing page per Zenolve public-data
feed (Apify pay-per-result actors), with schema, live sample data, code snippets,
FAQ, and JSON-LD structured data so search engines and AI answer-engines can
discover and cite each feed.

- **Live:** https://feeds.zenolve.com
- **Feeds source:** https://apify.com/zenolvepro

## Feeds (14 live)

Regulatory & compliance: FDA warning letters, CFPB enforcement actions, CPSC recalls ·
Finance & legal: SEC EDGAR latest filings, SCOTUS slip opinions, NBER working papers ·
Security: CERT/CC vulnerability notes, Android Security Bulletin CVEs ·
AI research: HuggingFace daily papers, arXiv cs.LG / cs.CV / cs.CL ·
Open source & devtools: F-Droid apps, Homebrew install analytics.

## How it's built

Static HTML, no framework. Pages are generated by
`auto_builder/builds/zenolve-feeds-site/generate.py` (catalog in
`feeds_config.py`), which pulls a live sample from each feed at build time and
renders the pages here — plus `sitemap.xml`, `robots.txt`, and an IndexNow key.
Deploys via GitHub Pages (root of `main`); `CNAME` points it at
`feeds.zenolve.com`.

To rebuild: run the generator, copy its `dist/` output here (minus `_preview/`),
commit, and push.
