What this feed does

HuggingFace's Daily Papers is the community-voted ranking of what matters in AI/ML research today - and it has no official bulk API. This feed scrapes the live page and returns each trending paper as structured JSON with its upvote count.

Structured, not scraped-by-you

Clean JSON with stable field names - no HTML parsing, no selectors to maintain when the site changes.

FAQ

Is there a free HuggingFace papers API?
There is no official bulk/list API for Daily Papers. This feed scrapes the live page and returns JSON at $0.01 per record.
How is the ranking determined?
It reflects HuggingFace's own community upvotes at the moment you call - a real-time signal of what researchers care about.
What does each record include?
title, upvotes, and the url of the paper's HuggingFace page.
How current is it?
Live - each call scrapes the page at request time, including today's papers.
Can an AI agent call this directly?
Yes - exposed as an MCP tool for native agent calls.
